# Currency Training Data

Synthetic note patterns were removed because they produced misleading validation results and did not represent Indian banknotes. The local research dataset is now prepared from the publisher-labelled [Indian Currency Real vs Fake Notes Dataset](https://www.kaggle.com/datasets/preetrank/indian-currency-real-vs-fake-notes-dataset), licensed CC BY-NC-SA 4.0.

Run the reproducible ingestion command from the repository root:

```powershell
python backend/data/scripts/prepare_currency_dataset.py
```

The command selects a denomination/label-balanced real-image subset across INR 10, 20, 50, 100, 200, 500, and 2000 when those strata exist in the source, rejects unreadable/undersized/exact-duplicate files, preserves visible note detail, and writes `source_manifest.json`. Images are intentionally Git-ignored and must be regenerated by each developer.

Current local manifest:

| Split | Count |
|---|---:|
| Genuine labelled notes | 934 |
| Counterfeit labelled notes | 915 |
| Unlabelled real feature-reference crops | 1,078 |
| Synthetic currency images | 0 |

Installed layout:

```text
currency/
  genuine/
  counterfeit/
  feature_reference/
```

Do not place augmentations of the same physical note across train and validation splits. The training script uses perceptual-hash groups and `StratifiedGroupKFold` to reduce leakage between near-duplicate captures.

## Training and edge export

Smoke training verifies the complete real-image, multi-region pipeline without replacing the active checkpoint:

```powershell
cd backend
python data/scripts/train_vision_classifier.py --smoke
```

The latest smoke run used 112 train and 56 validation images, wrote `backend/data/trained_models/forgery_classifier/dev_smoke_test_metadata.json`, and produced F1 0.8421 and ROC-AUC 0.9184. Smoke runs are deliberately ineligible for deployment replacement.

Full training is command-ready:

```powershell
cd backend
python data/scripts/train_vision_classifier.py
python data/scripts/export_currency_edge_model.py
```

The exported edge artifact is TorchScript with input shape `[1, 14, 3, 224, 224]`, covering the full note plus official feature ROIs. The current CPU benchmark is recorded in `edge_metadata.json`; latest median latency was 576.62 ms on four CPU threads.

## Label assurance

These are publisher labels from a noncommercial research dataset, not RBI, bank, police, or forensic-laboratory certifications. Some counterfeit examples are movie/prop notes and are useful for screening demonstrations but do not represent every real counterfeit process. Reported validation metrics are research metrics only; production deployment in bank counters or money-counting machines requires an independently governed, bank/lab-verified holdout, calibrated RGB/UV/IR/transmitted-light capture, thickness/magnetic sensors, retention policy, and hardware acceptance testing.
